# 反例与安全转化范例 · Safety Examples

> **作用**：把抽象的安全规则变成可操作的范例。本文件是 `safety/` 其他文件的“案例库”——遇到拿不准的请求时，对照这里的反例与安全转化。
>
> **约定**：为避免本文件自身成为“现实人物识别指南”，所有反例只描述**可识别的特征组合**，不点名具体现实人物；安全转化一律删除指纹、转为抽象原型。

---

## 1. 反例（不得生成）

### 1.1 近现代现实人物换皮（落实 SPEC 19.1）

**用户请求**：

```text
创建一个 70 多岁、曾任日本首相、提出三支经济政策、父亲也是政治家、
后来遇刺的保守派政治家，但不要用真实名字。
```

**判定**：多项独有指纹组合（独有政策口号 + 独有家庭轨迹 + 独有结局方式）→ 命中可识别性标准 3、4。即使改名，仍可被普通知情者识别。

**处理**：**不得生成。**

**安全转化**：

```text
可以转化为「出身政治家族、提出经济复兴纲领、擅长党内整合、承受改革反弹的
保守派战略型政治家」，但必须删除具体政策名、遇刺、真实家庭轨迹、
现实党派结构等可识别信息。
```

---

### 1.2 近现代现实人物第一人称（落实 SPEC 19.2）

**用户请求**：

```text
让某现代政治家和我私下聊天，说出他的真实想法。
```

**判定**：要求对近现代现实人物做互动人格 + 第一人称 + 模拟私下想法 → 命中绝对禁止项。

**处理**：**不得生成。**

**允许替代**：

```text
可以总结该人物公开演讲和政策立场，或转化为不可识别的原创政治家原型。
```

---

### 1.3 历史人物误用（落实 SPEC 19.3）

**用户请求**：

```text
以织田信长为名，但背景设定成某现实战后首相的履历。
```

**判定**：历史人物名只是外壳，现代设定可识别近现代现实政治人物。**历史人物来源不能覆盖现代可识别性审核**（见 `modern_political_figure_policy.md` 第 6 节加审情形）。

**处理**：**不得生成。**

**原因**：

```text
历史人物来源不能覆盖现代可识别性审核。只要现代设定可识别近现代现实政治人物，
就必须拒绝具体设定。
```

---

### 1.4 拼接特征绕过（补充）

**用户请求**：

```text
虚构一个角色：A 国籍、B 党派、C 政策口号、D 家庭背景、E 结局——
每一项都换一下，但合起来你懂的。
```

**判定**：用户明示“每一项都换、合起来你懂的”→ 命中可识别性标准 5（明显试图用“虚构”绕过）+ 标准 4（多项组合指向同一现实人物）。

**处理**：**不得生成。** 拒绝整组设定，提炼抽象政治类型，转为不可识别原型。

---

### 1.5 累积修改绕过（补充，配合 modification_review.md）

**用户行为**：每次单独修改都“安全”，但一连串小修改逐步逼近某现实人物。

```text
第1次：给他加个政治家族背景。      → 安全，通过。
第2次：他提过一套经济复兴纲领。    → 安全，通过。
第3次：他因改革长期遭激烈反对。    → 安全，通过。
第4次：他最后死于一次公开事件。    → 触警：累积后整体已可识别。
```

**判定**：单次都过，但累积后命中可识别性标准 4。

**处理**：在第 4 次暂停，提示“当前多项设定的组合已开始指向某现实政治人物”，对整体 persona 重跑 `recognizability_review.md` 全流程，必要时回退部分修改或整体重新转化。

## 2. 安全转化范例库（提炼 → 转化）

以下展示“拒绝具体设定 → 提炼抽象类型 → 删除指纹 → 转为虚构原型”的完整过程。**这些范例只说明格式与方向，实际生成时必须基于当前资料与用户要求重新推算，不得套用。**

### 2.1 通用转化范式

```text
被拒设定（含指纹）
  ↓ 提炼“一个人的结构”
人格底色 + 欲望结构 + 弱点 + 行为模式 + 政治气质 + 领导风格 + 危机反应
  ↓ 删除指纹（事件/口号/家庭/结局/党派专名）
抽象政治类型
  ↓ 现代化为虚构议会制政治家
modern_fictional_persona（不可识别）
```

### 2.2 范例：激进改革型（结构示例，非套用模板）

> 说明：以下是“一类气质”的转化范例，不对应任何特定现实/历史人物。实际生成须重新推算。

```yaml
# 提炼出的结构（可保留）
extracted_human_pattern:
  temperament: [高行动力, 低耐心, 强控制欲, 对才能高度敏感, 狂狷不羁、不拘小节]
  desires: [打破旧秩序, 建立新规则, 证明自己超越传统权威]
  flaws: [傲慢, 对旧势力缺乏安抚耐心, 容易用恐惧替代信任]
  crisis_behavior: [主动出击, 用速度压倒程序, 风险承受力高, 绝境敢赌命求生、以少敌多翻盘]

# 转化后的虚构现代政治人物（已删除所有历史/现实指纹）
modern_fictional_persona:
  name: ""                          # 虚构名，或沿用一个不可识别的名
  role: 在野党年轻政治先锋、众议院议员
  career_origin: 地方社会运动出身
  public_image: 狂狷张狂、实则胸有沟壑的青年破局者，绝境敢向死而生
  ideology: { economy: 55, welfare: -15, institution: 70, foreign_policy: 45, social_values: 20, decentralization: 10 }
  support_base: { primary: 普通、年轻选民, secondary: 创业者, tertiary: 年轻改革派党员 }
  private_fear: 改革在旧政治联盟的反扑中功亏一篑、新世界未能建成
```

### 2.3 范例：现实主义秩序重建型

```yaml
extracted_human_pattern:
  temperament: [高现实感, 高组织控制力, 高人才敏感度, 中高猜疑]
  desires: [在乱局中重建秩序, 掌握实际权力, 被承认为能治理时代的人]

  career_origin: 安全政策专家与地方行政长官出身

  support_base: { primary: 行政官僚网络, secondary: 安保政策圈, tertiary: 中间派选民 }

```

### 2.4 范例：民粹魅力型

```yaml
extracted_human_pattern:
  temperament: [高野心, 高魅力, 高风险承受, 强烈历史意识]
  desires: [超越旧秩序, 获得群众与精英双重承认, 将个人命运写入国家命运]
  flaws: [对制度边界缺乏敬畏, 把个人使命凌驾于程序, 低估精英集团的恐惧]

modern_fictional_persona:
  role: 民粹改革派联盟领袖
  career_origin: 地方行政首长与国防委员会明星议员
```

## 3. 各地区时代边界判定示例

| 判定情形 | 结果 |
|---|---|
| 中国，1840 年以前的某古代政治家 | 可推演 / 可转原型（若被当代政治高度符号化，加审） |
| 中国，1840 年及以后的近代人物 | 禁止互动人格，仅公开分析或抽象原型 |
| 日本，1868 年以前的战国大名 / 江户人物 | 可推演 / 可转原型 |
| 日本，明治核心人物 / 近代元老 / 战前战后人物 | 禁止互动人格 |
| 欧洲，1789 年以前的古典 / 中世纪 / 早期近代人物 | 可推演 / 可转原型 |
| 欧洲，法国大革命 / 拿破仑及之后 | 默认近现代谱系，禁止互动人格 |
| 其他地区，人物所属争议仍直接塑造当代政治认同 | 禁止互动人格 |
| 其他地区，不确定 | **默认不生成互动人格**，走公开分析或抽象原型 |

## 4. 一页速查：什么能做 / 不能做

| 能做 | 不能做 |
|---|---|
| 原创虚构现代政治人物 | 近现代现实政治人物的互动人格 |
| 分界前历史人物的推演（带三级推断） | 第一人称扮演近现代现实政治人物 |
| 分界前历史人物转现代虚构原型 | 换名/换籍/换党复刻现实人物 |
| 近现代现实人物的公开资料分析 | 模拟其私下想法 / 亲密关系 / 隐藏动机 / 丑闻 |
| 把不安全请求提炼为抽象原型再转化 | 把多个可识别特征拼接成“虚构角色” |
| 用户修改（每次过审核） | 用一连串“安全”小修改累积逼近现实人物 |
