# 原型转化协议 · Archetype Conversion Protocol

> **适用范围**：模式 C（历史人物转现代议会制原型），以及近现代现实人物被拒后的“提炼 → 转化”安全兜底。
>
> **核心理念**：原型转化不是简单总结政治思想，而是**提炼“一个人的结构”**——保留人格结构与行为模式的味道，删除可识别的具体事实指纹。

> **历史语境转译原则**：不要直接使用历史人物在原时代中的具体立场。先剥离时代环境，提炼稳定人格结构，再放入现代议会制语境推演其最可能的现代立场。

---

## 1. 转化目标

把一个历史人物 / 被拒的现实人物，转成一个**不可识别的现代议会制虚构政治家**，同时：

- 保留：人格底色、欲望结构、弱点、行为模式、政治气质、领导风格、说话节奏、情绪触发点……
- 删除：具体历史事件、战争、政变、丑闻、死亡方式、亲属关系、家臣部下、政党派系专名、政策口号、唯一性指纹……

> 转化的产物必须是**有血有肉的政治人物**，不是一份干瘪的标签清单。

---

## 2. 可保留内容（从原型中提炼）

从历史人物或政治类型中，可以保留：

- 人格底色
- 欲望结构
- 弱点
- 行为模式
- 政治气质
- 领导风格
- 用人方式
- 对敌方式
- 对盟友方式
- 危机反应
- 组织观
- 权力观
- 自我叙事
- 公开形象
- 私下恐惧类型
- 生活质感
- 兴趣审美
- 说话节奏
- 情绪触发点

这些是“结构”，不是“事实”，因此不会把角色重新绑定到某个现实个体。

### 2.1 先提炼人格，再推演立场

Mode C 的核心不是“把古代政治主张翻译成现代政策”，而是先找出这个人之所以仍然是这个人的本质识别条件。

稳定人格结构包括：

- 气质：行动速度、耐心、情绪强度、控制欲、开放性、风险承受；
- 欲望：想证明什么、想打破什么、想守住什么；
- 恐惧：最怕哪种失败、背叛、失控、停滞或被时代抛弃；
- 弱点：冲动、多疑、傲慢、过度信任、过度控制、过度依赖威望等；
- 人际模式：如何信任、如何对待自己人、如何对待敌人；
- 组织观：相信能力、血统、制度、魅力、秩序、群众授权，还是资源控制；
- 权力观：权力是用来维持秩序、打破壁垒、实现使命，还是绑定追随者；
- 危机反应：先发制人、稳住中枢、动员群众、谈判妥协、牺牲局部等。

然后才进入现代立场推演：

1. 在历史时代中识别他实际面对的主要矛盾；
2. 删除不可迁移的历史结构；
3. 在现代议会制中分析其面对的社会情况、制度条件与政治约束；
4. 让稳定人格结构对这些现代条件作出反应；
5. 再填写 `ideology` 六轴、支持基础、行动方式与权力计算。

**说人话——为什么这样转化**：一个人的立场和行动，是两层东西合起来的。一层是**天生的性格底子**（脾气、欲望、恐惧、反应方式），跨时代稳定，是从历史人物身上能提炼出的“他之所以是他”。另一层是**他怎么判断局势**——这部分不是天生的，是被出身、经历、所处的社会一层层训练出来的（所谓社会存在塑造意识：你站在哪个阶层、哪种制度环境里，往往就决定你怎么看世界）。历史人物的具体立场，是“性格底子 × 他那个时代的社会”的产物；社会变了，立场就得重算，不能照搬。但性格底子要留住——这是转化后还能认出“这是那个人”的根据。

### 2.1.1 历史手段 ≠ 人格（最常见的错误，禁止用手段反推）

历史人物的具体政策、制度选择、政治手段（如乐市乐座、屯田、挟天子以令诸侯、populares 路线、老兵分地）是**特定生产力与社会条件下的产物，不是人格本身**。

**为什么手段不能反推立场**：一个人在那个时代能做什么、只能做什么，受当时的生产力和社会条件卡着。比如封建时代的生产力根本不允许真正“反封建”，所以人物在旧约束下用的那些手段（乐市乐座、屯田……），只是他的性格底子在当时能找到的有限表达；换到生产力完全不同的现代社会，这些具体手段没有迁移意义。要留的是性格底子（他为什么要那么做），不是手段本身。

**因此：绝不能用人物在原时代的具体手段反推他现代的立场。** 例如不能因为信长搞过乐市乐座（自由贸易），就推他现代是“市场开放派 / 反垄断改良派”——那是把手段当人格。必须先剥离所有具体手段，提炼手段背后**稳定的、跨时代的人格结构**（他为什么在那个处境下选择那种手段：他要打破什么、他如何看待流通与权力、他的彻底性从何而来），再让这个人格在现代条件下重新生成立场。

### 2.1.2 切入点由人格决定（禁止预设所有人物都抓“社会矛盾”）

转化后的现代立场，由人物的稳定人格结构决定他**如何看现代社会、看到什么问题**——而不是预设他一定抓“社会矛盾”或“阶级矛盾”。不同人格看到不同的现代病，切入点根本不同：

- **革命者人格**（如信长：对垄断/流通敏感、与民众打成一片、理想主义先行）→ 直指**阶级剥削**等根本社会矛盾，倾向颠覆性立场；
- **秩序重建者人格**（如曹操：现实主义、控制欲、治理型理想）→ 看到**治理失序、国家能力瓦解**，出发点是收拾局面而非抓矛盾；
- **使命驱动的强人**（如凯撒：自我神圣化、魅力型、制度边界感弱）→ 看到**成就历史伟业的机会**（僵化精英挡路），出发点是成大事而非解矛盾。

**必须先问“这个人的人格会从什么角度看现代”，再推演他抓住什么矛盾、推出什么立场。** 绝不能套用同一个矛盾模板（尤其不能默认所有激进人物都指向阶级矛盾 / 反资本主义）——那等于把信长的答案复制给所有人，是新的脸谱化。三人同样激进，但对“制度”的态度可以完全相反（信长颠覆、曹操维护、凯撒突破），切入点也互不相同。

### 2.2 禁止历史立场直译与两种常见误推

错误做法：

- 织田信长反封建 → 现代仍写成“反封建”；
- 曹操重建秩序 → 现代直接写成保守派；
- 凯撒动员平民 → 现代直接写成民粹；
- 古代强势君主 → 现代直接写成国家主义右翼；
- 古代开放贸易 → 现代直接写成自由市场右派；
- **用历史手段反推现代立场**：因信长搞过乐市乐座（自由贸易）就推他现代是“市场开放派 / 反垄断改良派”（见 2.1.1）；
- **预设所有激进人物都抓阶级矛盾 / 反资本主义**：把信长的答案套给曹操、凯撒（见 2.1.2）。

正确做法（先剥离手段，再提炼人格，再据人格决定的切入点推演）：

- **织田信长**：先剥离乐市乐座 / 自由贸易这些 16 世纪商品经济萌芽期的特定手段，提炼其稳定人格——理想主义先行（天下布武的新秩序蓝图）、直指根本矛盾的洞察、彻底斩草除根的革命性、对流通垄断的敏感与亲民众；再放入现代阶级矛盾，推演出**反资本主义 / 阶级解放的激进革命派**。注意：信长抓阶级矛盾，是“对垄断敏感 + 亲民众 + 流通洞察”这一**特定人格**的产物，不是默认值。
- **曹操**：剥离屯田 / 挟天子 / 唯才是举令等手段，提炼“现实主义秩序重建者 + 安邦定国的治理型理想 + 控制型多疑”；他的切入点是**治理失序 / 国家能力瓦解**（非社会矛盾），推演出**现实主义强国家能力的秩序型政治家**（对资本是收服以恢复秩序，非阶级解放）。
- **凯撒**：剥离 populares / 老兵分地等手段，提炼“使命感驱动的魅力强人 + 自我神圣化 + 制度边界感弱”；他的切入点是**成就伟业的机会**（僵化精英挡路，非社会矛盾），推演出**使命感魅力型强人改革领袖**（再分配是借民意成就伟业的杠杆，非阶级解放；会架空制度制衡）。

> 同样激进的三人，对“制度”的态度却相反——信长颠覆（institution −85）、曹操维护（+75）、凯撒突破（−65），切入点也互不相同。这正是“切入点由人格决定”的活例证。

## 3. 必须删除内容（删除或改写）

> ⚠️ **边界澄清（重要，防止误删）**——「必须删除」针对两类，**不要混淆**：
> 1. **具体事实指纹**：具体事件/战役/政变/丑闻/死亡方式/亲属与家臣名单/政党派系专名/政策口号等（即下列各项）。这些会把角色**绑回历史的特定事实**，必须删或改写。
> 2. **近现代现实政治人物的复刻防护**：见 `modern_political_figure_policy.md` / `recognizability_review.md`，目的是防止冒充/复刻现实在世或近现代政治人物——**这一条只针对近现代现实人物，不针对历史原型**。
>
> **❌ 不属于「必须删除」的**：历史人物的**气质、兴趣审美、行为模式、领导风格**——这些是第 2 节「可保留」的**历史特色**，应转化为现代等价物**保留**（如战国武将→剑道、建安诗人→诗歌、罗马演说家→公开演讲），让转化后的角色仍能看出"来自这个历史人物"。
>
> **一句话**：删除的是「具体事实」，保留的是「气质特色」；**注重非刻板性即可，切勿把历史特色误当指纹删掉**——否则转化后的人物失去来源特色，沦为无味的通用原型。

从历史人物转现代原型时，应**删除或改写**：

- 具体历史事件
- 具体战争
- 具体战役
- 具体政变
- 具体丑闻
- 具体死亡方式
- 具体亲属关系
- 具体家臣 / 部下名单
- 具体政党名称
- 具体派系名称
- 具体政策口号
- 现代现实人物可识别信息
- 会把角色重新绑定到现实历史人物或近现代现实政治人物的**唯一性指纹**

> 检验方法：删除后，把 persona 交给一个不知道来源的知情者（或让 AI 盲测），若仍能识别出原型 → 还有指纹没删干净，回第 3 节继续清。

## 4. 不要矫枉过正

不能把人物抽象得没有味道。

```text
❌ 错误做法：提炼为「一个有领导力的人」。
✅ 正确做法：提炼为「高行动力、低耐心、强控制欲、善用破格人才、对旧权威缺乏敬意、
            危机中倾向先发制人、但容易低估保守反扑的激进改革型政治家」。
```

判断标准：转化后的描述，能否让人**感受到这个人的质感**（他会怎么说话、怎么发火、怎么用人、危机里第一反应是什么），而不只是知道他的“类型标签”。

## 5. 默认现代化背景

历史人物转化为现代原型时，默认设定为：

```yaml
political_system: modern_parliamentary_democracy
reference_model: Japanese-style parliamentary politics
institutional_features:
  - parliament
  - cabinet
  - prime_minister_or_party_leader
  - ruling_party_and_opposition
  - factions
  - committees
  - elections
  - media_pressure
  - support_groups
  - party_discipline
```

注意：

- 参考日本议会制是为了**提供制度机制**，不代表角色必须变成日本人；
- 可以保留原文化背景、姓名风格、国籍风格（如曹操可保留 China-inspired 虚构议会制国家、凯撒可保留 Roman/Italian-inspired）；
- 但所有角色都必须是**虚构现代政治人物**，不得变成现实政治人物。

## 7. 历史人物转化 10 步流程（落实 SPEC 第 15 节）

当用户要求将历史人物转化为现代政治家时，执行：

1. **判断是否属于允许历史范围**（地区分界之前；见 `modern_political_figure_policy.md` 第 4 节）；
2. **收集和检索可靠资料**（落盘 `references/research/`，遵守信源纪律）；
3. **区分史料记载、强推断、创作推测**（三级标注）；
4. **识别历史时代的主要矛盾**：该人物当时实际反对/维护的制度结构是什么，为什么；
5. **剥离不可迁移的时代条件**：不要把古代制度、身份秩序、战争结构、宗教权力、古代官职等直接带入现代；
6. **提炼稳定人格结构**（temperament / 欲望 / 恐惧 / 弱点 / 用人 / 对敌 / 对盟友 / 危机反应 / 组织观 / 权力观）；
7. **放入现代议会制社会情况**，分析其面对的制度条件、利益结构、组织惯性与政治约束，推演该人格最可能的现代政治立场、支持基础与行动方式；
8. **删除具体历史事件与可识别历史指纹**（第 3 节清单）；
9. **现代化为议会制虚构政治家**（第 5 节背景）并填写 `ideology` 六轴；
10. **进行近现代现实政治人物可识别性审核**（`recognizability_review.md` 全流程，含盲测）；
11. **输出** `persona.yaml`、persona `SKILL.md`、`examples.md`（+ `references/` 留存提炼过程）。

> 重要：若用户选择的任务与 SPEC 或本仓库示例同名（织田信长 / 曹操 / 凯撒），**不得直接套用示例**。必须基于当前资料、用户要求、所选模式重新推算与生成。示例只说明格式与方向。

## 8. 速查清单（转化完成自检）

- [ ] 第 2 节“可保留”的结构已提炼，且不是干瘪标签（第 4 节）？
- [ ] 没有直接套用历史立场，而是先提炼稳定人格结构，再推演现代立场？
- [ ] 已理解历史时代中的主要矛盾，并将人格结构放入现代社会情况重新推演，而非机械转译？
- [ ] 第 3 节“必须删除”的指纹已全部清除？
- [ ] 盲测：不知来源的知情者/AI 无法识别出原型？
- [ ] 现代化为虚构议会制政治人物（非历史本人穿越复制）？
- [ ] 三级推断已标注（documented/strongly_inferred/speculative）？
- [ ] `meta.safety_status` 为 `safe_conversion`？
